# database/

Каталог `database/` содержит только артефакты PostgreSQL и инициализации БД.

## Содержимое

- `postgres/schema.sql` — SQL-схема и индексы для таблиц ANPR.

## Важно

- Здесь **не** размещается Python-код приложения и бизнес-логика.
- Runtime-код доступа к БД остаётся в пакетах приложения (`anpr/`, `apps/`, `packages/`).
